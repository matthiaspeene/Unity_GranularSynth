# Changelog

All notable changes to the granular synth will be documented in this file.

## [0.1.0] - 2026-07-10
### Initial Commit
- Scriptable Object Granular Synth:
- Controls for: Rate, Duration, Playback Position, Pitch, Pan, Gain and Grain Shape.
- Anti phasing toggle
- Stereo File Support

## [0.2.0] - 2026-07-23
### Modulation Matrix
- Added a modulation matrix to make dynamic parameters more easily configurable.
